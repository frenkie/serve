Welcome to the example server

This page is not intended to be called directly, but through the example server.

So fire it up by navigating to /examples/server and execute node server.

From there you can try out Serve.

Examples

serve.raw()

  • index.html (as a fallback for requesting a directory)
  • js/test.js
  • template/mustache.html

serve.json()

  • js/test.js?json=data
  • template/mustache.html?json=data

serve.jsonp()

  • js/test.js?json=notdefault&callback=example
  • js/test.js?callback=example (jsonAttribute defaults to 'data')
  • template/mustache.html?json=notdefault&callback=example
  • template/mustache.html?callback=example (jsonAttribute defaults to 'data')
